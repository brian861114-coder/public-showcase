波動是物理學中最核心的能量傳遞機制之一。從海浪拍打岸邊、吉他弦的振動、到地震波穿越地殼，波將能量從一處傳送到另一處，卻不傳送物質本身。本章建立在第十五章振盪的基礎上，將簡諧運動的概念推廣到連續介質中，系統性地介紹機械波的行為。

你將從波的基本特性（波長、頻率、振幅、波速）開始，學習如何用數學波函數精確描述波，接著探討波速如何由介質的彈性與慣性決定，理解波的能量如何正比於振幅平方，最後進入波的疊加——從干涉現象到駐波與共振，這些概念將成為後續學習聲音、電磁波與量子力學的基礎。

⚡ 核心心法：波速 v = λf 是所有波動的統一關係式，但波速本身只取決於介質特性，不由頻率或波長決定。這一點是初學者最常搞混的地方。

C16.1 行進波（Traveling Waves）

波的基本特性

波（wave）是一種擾動的傳播——擾動從產生處向外移動，將能量與動量從一點傳遞到另一點，但不傳送物質。波有三種基本類型：機械波需要介質（如水波、聲波、地震波），由牛頓定律支配；電磁波由電場與磁場振盪構成，不需介質，可在真空中以光速傳播；物質波是量子力學的核心概念——所有物質粒子都具有波動性，由德布羅意（de Broglie）於 1924 年首先提出。

💡 三種波的對照：
類型需要介質？範例物理基礎
機械波是水波、聲波、地震波牛頓力學
電磁波否光、X 光、無線電波Maxwell 方程式
物質波否電子繞射、中子干涉量子力學

橫波與縱波

機械波依介質擾動方向與波傳播方向的關係分為兩種：橫波（transverse wave）的介質擾動方向垂直於傳播方向，如弦上的波、水面漣漪。橫波有波峰（crest，最高點）與波谷（trough，最低點）。縱波（longitudinal wave）的介質擾動方向平行於傳播方向，如聲波在空氣中的疏密變化、彈簧的壓縮波。

🟢 橫波（Transverse）

介質振動 ⟂ 傳播方向
例：弦波、水面波、S 地震波
有波峰與波谷

🔵 縱波（Longitudinal）

介質振動 ∥ 傳播方向
例：聲波、P 地震波
有疏部與密部

波的關鍵參數

描述一個簡諧波需要以下五個基本量：

  • 波長（wavelength）λ：相鄰兩個相同相位點之間的距離，通常取兩個相鄰波峰或波谷
  • 振幅（amplitude）A：介質偏離平衡位置的最大位移
  • 週期（period）T：完成一次完整振盪所需的時間，也是單一波長通過固定點所需時間
  • 頻率（frequency）f：單位時間內通過固定點的波數，f = 1/T，單位為赫茲（Hz）
  • 波速（wave speed）v：擾動在介質中傳播的速度
$$v = \frac{\lambda}{T} = \lambda f \tag{16.1}$$
⚡ 關鍵：v = λf 是所有波（機械波、聲波、光波）都適用的基本關係式。但波速 v 本身取決於介質的特性，而非由頻率或波長決定。當波從一種介質進入另一種介質時，v 會改變，導致 λ 改變，但 f 保持不變。
📐 例題 16.1 — 弦上的波
一條 30.00 m 的弦，一端固定在牆上。學生以 2.00 Hz 的頻率上下移動自由端，最大位移 20.00 cm。第一道波在 6.00 s 後到達牆面。

(a) 波速：v = Δx/Δt = 30.00/6.00 = 5.00 m/s
(b) 週期：T = 1/f = 1/2.00 = 0.500 s
(c) 波長：λ = v/f = 5.00/2.00 = 2.50 m

E16.2 波的數學描述（Mathematics of Waves）

簡諧波函數

當波通過介質時，介質中每個質點都在做簡諧運動。我們可以用一個數學函數——波函數（wave function）——來描述任意位置 x 在任意時間 t 的介質位移 y(x,t)。對一個沿 +x 方向以波速 v 傳播的正弦波：

$$y(x,t) = A \sin(kx - \omega t + \phi) \tag{16.4}$$

其中各符號的意義如下表：

符號名稱定義單位
A振幅最大位移m
k波數（wave number）k = 2π/λrad/m
ω角頻率ω = 2π/T = 2πfrad/s
φ初始相位t=0, x=0 時的相位rad
kx − ωt + φ相位（phase）決定振盪狀態rad

波速與波數、角頻率的關係為：

$$v = \frac{\omega}{k} \tag{16.5}$$
💡 波函數的物理直覺：波函數 y(x,t) = A sin(kx − ωt + φ) 同時描述了空間中的形狀（由 kx 控制——固定某時刻 t，它是 x 的正弦函數，週期為 λ）和時間中的振盪（由 ωt 控制——固定某位置 x，它是 t 的正弦函數，週期為 T）。負號（kx − ωt）表示波向 +x 方向傳播；若為正號（kx + ωt），則向 −x 方向傳播。
📐 例題 16.3 — 從波函數讀取波形參數
已知波函數 y(x,t) = 0.2 sin(6.28x − 1.57t) m（x 單位 m，t 單位 s）：

A = 0.2 m，k = 6.28 rad/m，ω = 1.57 rad/s
λ = 2π/k = 2π/6.28 = 1.00 m
T = 2π/ω = 2π/1.57 = 4.00 s，f = 1/T = 0.25 Hz
v = ω/k = 1.57/6.28 = 0.25 m/s（向 +x 方向）

線性波動方程式

所有簡諧波函數都滿足一個更基本的方程式——線性波動方程式（linear wave equation），它是物理學與工程學中最重要的方程式之一：

$$\frac{\partial^2 y}{\partial x^2} = \frac{1}{v^2}\frac{\partial^2 y}{\partial t^2} \tag{16.6}$$

任何滿足此方程式的函數都是線性波函數。一個非常重要的性質是：若 y₁ 和 y₂ 各自滿足波動方程式，則它們的任意線性組合 Ay₁ + By₂ 也滿足——這就是後續要討論的疊加原理（principle of superposition）的數學基礎。

⚡ 線性 vs 非線性：疊加原理只對線性波成立。當振幅過大時，介質的回復力不再是線性的（例如巨浪破碎、大振幅聲波產生衝擊波），就不能用簡單疊加處理。本章只討論線性波。

F16.3 弦上的波速（Wave Speed on a Stretched String）

波速取決於介質特性

對一條張緊的弦，波速由兩個因素決定：張力（tension）FT——代表介質的彈性回復力，和線密度（linear density）μ = m/L——代表介質的慣性。透過對弦上一個小質量元素做牛頓第二定律分析，可以推導出：

$$v = \sqrt{\frac{F_T}{\mu}} \tag{16.8}$$

其中 FT 為弦的張力（單位 N），μ = m/L 為單位長度的質量（單位 kg/m）。這個關係式的物理意義非常直觀：張力越大（彈性回復力越強），波速越快；線密度越大（慣性越大），波速越慢。

📖 推導過程（牛頓第二定律分析）

考慮弦上一個小質量元素 Δm = μΔx。當弦被撥動產生橫波時，兩端張力的 y 分量不相等，產生淨回復力：Fnet,y = FT(∂y/∂x|x₂ − ∂y/∂x|x₁)。

由牛頓第二定律：Fnet,y = (μΔx)ay = (μΔx)(∂²y/∂t²)。

取 Δx → 0 的極限，得到：μ(∂²y/∂t²) = FT(∂²y/∂x²)，即 ∂²y/∂x² = (μ/FT)(∂²y/∂t²)。

與線性波動方程式 ∂²y/∂x² = (1/v²)(∂²y/∂t²) 對比，得 v = √(FT/μ)。

📐 例題 16.5 — 吉他弦的波速
高音 E 弦的線密度 μhigh E = 3.09×10⁻⁴ kg/m，張力 56.40 N：
v = √(56.40 / 3.09×10⁻⁴) = √(1.825×10⁵) ≈ 427 m/s

低音 E 弦的 μ 約為高音弦的 20 倍。要保持相同波速，張力需增加約 20 倍：
FT ≈ 20 × 56.40 ≈ 1128 N → 實際計算得約 1128 N。
→ 這解釋了為什麼低音弦比高音弦更粗、張力更大。

流體中的縱波波速

一般來說，任何介質中的波速都可寫成 √(彈性參數/慣性參數) 的形式。對流體（液體或氣體）中的縱波（如聲波）：

$$v = \sqrt{\frac{B}{\rho}} \tag{16.9}$$

其中 B 為體積模量（bulk modulus，代表流體的抗壓縮性），ρ 為密度。

💡 吉他調音的物理：吉他六根弦（E₂、A₂、D₃、G₃、B₃、E₄）的基本頻率分別為 82.41、110.00、146.83、196.00、246.94、329.63 Hz。弦長固定時，頻率由波速決定：f = v/λ。要改變音高，可調整張力（調音旋鈕）或改變有效長度（按弦）。粗弦的 μ 大→波速小→頻率低。

R16.4 波的能量與功率（Energy and Power of a Wave）

能量在波中的分布

波在介質中傳播時，每個質量元素都在做簡諧運動，因此同時具有動能與位能。對一個弦上的正弦波 y(x,t) = A sin(kx − ωt)，經過對單一波長範圍的積分，可以得到：

  • 單一波長內的動能 = 單一波長內的位能
  • 單一波長內的總能量：
$$E_\lambda = \frac{1}{2}\mu \omega^2 A^2 \lambda \tag{16.10}$$

時間平均功率

波的功率定義為單位時間內通過固定點的能量。對正弦波，時間平均功率為單一波長的能量除以週期：

$$P_{\text{avg}} = \frac{1}{2}\mu \omega^2 A^2 v \tag{16.11}$$
⚡ 功率正比於振幅平方與頻率平方：P ∝ A² 且 P ∝ ω²（即 P ∝ f²）。這意味著：若振幅加倍，功率變為 4 倍；若頻率加倍，功率也變為 4 倍。這個關係對所有機械波都成立，包括聲波——這就是為什麼大音量（大振幅）和高音調（高頻率）都需要更多能量。

波的強度

強度（intensity）I定義為單位面積通過的功率：I = P/A，SI 單位為 W/m²。對於從點源向三維空間發散的球面波，波前表面積隨半徑平方增長（A = 4πr²），因此：

$$I = \frac{P}{4\pi r^2} \tag{16.12}$$

這解釋了為什麼離波源越遠，聲音越小——能量守恆，但分布在越來越大的面積上。

📐 例題 16.6 — 弦振動器的功率
一條長 2.00 m、質量 70.00 g 的弦，張力 90.0 N，以頻率 60 Hz、振幅 4.00 cm 振動：

μ = 0.0700/2.00 = 0.0350 kg/m
v = √(90.0/0.0350) = 50.7 m/s
ω = 2π×60 = 377 rad/s
Pavg = ½ × 0.0350 × (377)² × (0.04)² × 50.7 ≈ 201 W

J16.5 波的干涉（Interference of Waves）

反射與邊界條件

當波遇到介質邊界時會發生反射。反射波的相位取決於邊界條件：

  • 固定端（fixed end）：反射波相位反轉 180°（π rad）——波峰反射為波谷。原因：牛頓第三定律——弦對牆施力向上，牆對弦施力向下。
  • 自由端（free end）：反射波與入射波同相——波峰反射為波峰。端點可自由移動，不受約束。
  • 介質變化：從低密度到高密度→反射波反相；從高密度到低密度→反射波同相。

疊加原理與干涉

疊加原理（principle of superposition）：當兩個或多個波在同一介質中相遇時，每個點的合成位移等於各波在該點造成位移的代數和。這源自波動方程式的線性性質。

🟢 建設性干涉

兩個波同相（相位差 = 0, 2π, 4π...）
波峰對波峰、波谷對波谷
合成振幅 = A₁ + A₂（若 A₁=A₂，則為 2A）
波長不變

🔴 破壞性干涉

兩個波反相（相位差 = π, 3π, 5π...）
波峰對波谷
合成振幅 = |A₁ − A₂|（若 A₁=A₂，則為 0，完全抵消）

相位差造成的干涉結果

考慮兩個完全相同（同 A、同 k、同 ω）但相位差為 φ 的正弦波：

$$y_R(x,t) = \left[2A\cos\left(\frac{\phi}{2}\right)\right] \sin\left(kx - \omega t + \frac{\phi}{2}\right) \tag{16.13}$$

合成波仍為正弦波，但振幅變為 2A·cos(φ/2)。

  • φ = 0 → 振幅 = 2A（完全建設性干涉）
  • φ = π → 振幅 = 0（完全破壞性干涉）
  • φ = π/2 → 振幅 = √2·A ≈ 1.41A
💡 干涉在日常生活中的例子：降噪耳機利用破壞性干涉——麥克風拾取環境噪音，耳機產生一個反相聲波，兩個波疊加後振幅歸零，你便聽不到噪音。音樂廳的聲學設計則利用建設性干涉來增強特定頻率的聲音。兩顆石頭丟入池塘產生的圓形漣漪互相穿過而不改變形狀，也是疊加原理的體現。

Q16.6 駐波與共振（Standing Waves and Resonance）

駐波的形成

當兩個完全相同、行進方向相反的波在同一介質中疊加時，結果看起來像一個「站在原地振動」的波——這就是駐波（standing wave）。考慮 y₁ = A sin(kx − ωt) 向右傳播，y₂ = A sin(kx + ωt) 向左傳播：

$$y(x,t) = y_1 + y_2 = 2A \sin(kx) \cos(\omega t) \tag{16.14}$$

這個波函數的關鍵特徵：空間部分 sin(kx) 與時間部分 cos(ωt) 完全分離。某些位置永遠靜止（節點，nodes），某些位置振盪幅度最大（腹點，antinodes）。

位置條件特性
節點 (Node)x = n·λ/2, n=0,1,2,...位移恆為零，完全不動
腹點 (Antinode)x = (2n+1)·λ/4, n=0,1,2,...位移在 ±2A 之間振盪

兩端固定的弦——共振模態

一條長度 L 的弦兩端固定時，兩端必然是節點（位移為零）。這意味著弦長必須等於半波長的整數倍：L = n·λn/2。對應的特徵波長與特徵頻率為：

$$\lambda_n = \frac{2L}{n}, \quad f_n = n\frac{v}{2L} = n f_1, \quad n = 1,2,3,\ldots \tag{16.15}$$

其中 f₁ = v/(2L) 稱為基頻（fundamental frequency），也稱為第一諧波。f₂ = 2f₁ 為第二諧波（第一泛音），f₃ = 3f₁ 為第三諧波（第二泛音），以此類推。所有諧波都是基頻的整數倍。

📖 駐波的推導（三角恆等式法）

兩個反向行進的相同波：y₁ = A sin(kx − ωt)，y₂ = A sin(kx + ωt)。

利用三角恆等式 sin(α) + sin(β) = 2 sin((α+β)/2) cos((α−β)/2)：

y = A[sin(kx−ωt) + sin(kx+ωt)] = 2A sin(kx) cos(ωt)。

節點條件：sin(kx) = 0 → kx = nπ → x = nλ/2。

腹點條件：|sin(kx)| = 1 → kx = (2n+1)π/2 → x = (2n+1)λ/4。

兩端固定的邊界條件：x=0 和 x=L 都是節點 → L = nλ/2 → λn = 2L/n → fn = nv/(2L)。

共振（Resonance）

當一個系統被以其自然頻率驅動時，即使驅動力很小，也能產生大振幅的振盪——這就是共振。在波的語境下，共振頻率就是系統的特徵頻率（f₁, f₂, f₃,...）。

⚡ 共振的雙面性：共振可以是美妙的（樂器的共鳴箱放大聲音、微波爐以水分子共振頻率加熱），也可以是災難性的（1940 年 Tacoma Narrows 橋因風引起的共振而倒塌、建築物在地震中的共振破壞）。工程設計中必須仔細考慮共振頻率。
📐 吉他弦的共振頻率計算
一根吉他弦長 L = 0.65 m，波速 v = 400 m/s：
基頻 f₁ = v/(2L) = 400/(2×0.65) = 307.7 Hz（接近 E₄ 的 329.6 Hz，可透過調音吻合）
第二諧波 f₂ = 2×307.7 = 615.4 Hz
第三諧波 f₃ = 923.1 Hz
→ 按弦改變 L 可演奏不同音高。例如按在弦長一半處，L 減半 → f 加倍 → 高八度。

  1. 波的基本關係式 v = λf 適用於所有類型的波。波速由介質的彈性與慣性決定，頻率由波源決定，波長隨之調整。
  2. 三種波——機械波（需介質，由牛頓力學支配）、電磁波（不需介質，由 Maxwell 方程式支配）、物質波（量子力學，所有粒子皆有波動性）。
  3. 橫波（擾動 ⟂ 傳播，如弦波）與縱波（擾動 ∥ 傳播，如聲波）是機械波的兩種基本形態。
  4. 波函數 y(x,t) = A sin(kx − ωt + φ) 完整描述簡諧波，其中 k = 2π/λ 為波數，ω = 2πf 為角頻率。
  5. 線性波動方程式 ∂²y/∂x² = (1/v²)∂²y/∂t² 是所有線性波必須滿足的偏微分方程，疊加原理由此導出。
  6. 弦上的波速 v = √(FT/μ)：張力越大越快，線密度越大越慢。
  7. 波的能量正比於振幅平方與頻率平方：P = ½μω²A²v。
  8. 波的強度 I = P/A，球面波的強度以 I ∝ 1/r² 衰減。
  9. 疊加原理：多個波同時存在時，合成位移為各波位移的代數和。這是干涉現象的基礎。
  10. 建設性干涉（同相，振幅相加）與破壞性干涉（反相，振幅相減）是所有波動現象的核心機制。
  11. 反射相位取決於邊界條件：固定端反相（π），自由端同相。
  12. 駐波 y = 2A sin(kx) cos(ωt)：由兩個反向行進的相同波疊加形成，有固定不動的節點和振幅最大的腹點。
  13. 兩端固定的弦，其共振模態為 fn = n·v/(2L) = n·f₁，諧波為基頻的整數倍。
  14. 共振發生在驅動頻率等於系統自然頻率時，可在微小驅動力下產生大振幅。
  15. 這些波的基礎概念將在後續章節中延伸至聲音（Ch17）、電磁波、光學與量子力學。
